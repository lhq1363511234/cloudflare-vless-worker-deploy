nthlink

          
主打“一键就能用”，Win / macOS / 安卓端无需复杂配置，不限速。节点少，晚高峰略挤。

老牌免费客户端，注册后可用一定额度。免费层有流量/速度限制，部分节点易被针对性封锁。

Cloudflare WARP

          
1.1.1.1 客户端一键连。若开通 CF Zero Trust，可获更稳定高速流量。系统级 VPN，适合兜底。
